# SQL-task1

This repository contains SQL practice questions and solutions based on Employee and Department tables.

I created this project to improve my SQL skills and gain hands-on experience with concepts commonly used in Data Analyst roles and technical interviews.

## What I Worked On

* Creating tables and inserting data
* Aggregate functions
* INNER JOIN and LEFT JOIN
* GROUP BY and HAVING
* Subqueries
* Finding the highest and second-highest salaries
* Comparing employee salaries with department averages
* Finding duplicate records
* Removing duplicate records using ROW_NUMBER()

## Files Included

* `employee_dataset.sql` – Table creation and sample data
* `sql_queries.sql` – SQL queries and solutions
* `README.md` – Project documentation

## SQL Questions Solved

1. Top 5 highest salary employees
2. Department-wise employee count
3. Find second highest salary
4. Employees earning more than their department average salary
5. INNER JOIN
6. LEFT JOIN
7. GROUP BY with HAVING
8. Employees hired in the last 6 months
9. Find duplicate records
10. Remove duplicate records

## What I Learned

This project helped me understand how SQL is used to analyze data, work with multiple tables, summarize information, and solve real-world business questions.
